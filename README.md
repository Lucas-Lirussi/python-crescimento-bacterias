# Análise de Crescimento de Colônias Bacterianas em Python

Este repositório contém exercícios desenvolvidos em Python com o objetivo de
praticar lógica de programação e conceitos iniciais de Data Science.

Os códigos analisam o crescimento de colônias bacterianas a partir de dados
simulados, utilizando cálculos percentuais e estruturas de repetição.

---

## Projeto 1 — Crescimento Percentual Diário

Calcula a variação percentual diária do número de bactérias (em milhares)
com base em uma lista de valores.

### Conceitos utilizados:
- Listas
- Laço `for`
- Acesso por índice
- Cálculo de variação percentual

---

## Projeto 2 — Comparação de Crescimento entre Colônias

Simula o crescimento de duas colônias bacterianas (A e B) com taxas diferentes
e calcula quantos dias são necessários para que a colônia A ultrapasse ou iguale
a colônia B.

### Conceitos utilizados:
- Variáveis
- Laço `while`
- Operadores matemáticos
- Simulação de crescimento

---

## Tecnologias
- Python 3

## Observação
Os dados utilizados são fictícios e têm finalidade educacional.
